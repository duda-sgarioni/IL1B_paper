# Analysis of IL1B variants in Congenital Zika Syndrome

R scripts for statistical analysis reported in the manuscript

## Study overview
Case-control study investigating the association between *IL1B* variants - rs4848306 (-3737G>A), rs1143623 (-1464C>G), rs16944 (-511A>G), and rs1143627 (-31G>A) - and Congenital Zika Syndrome (CZS) susceptibility in a cohort of Brazilian children with prenatal exposure to ZIKV

## Scripts
| Script | Description |
|--------|-------------|
| `clinical_analysis.R` | Sociodemographic and clinical characterization of cases and controls |
| `genetic_analysis.R` | Hardy-Weinberg equilibrium, allelic frequency comparisons, genotypic association, linkage disequilibrium, haplotype analysis, and logistic regression |

## Data Availability
Data are not publicaly available due to ethical restrictions
